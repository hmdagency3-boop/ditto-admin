# ملفات SQL - Supabase

هذا المجلد يحتوي على جميع أكواد قاعدة البيانات مرتبة حسب التعديلات.

## كيفية الاستخدام
1. افتح مشروعك على [supabase.com](https://supabase.com)
2. من القائمة الجانبية اضغط **SQL Editor**
3. شغّل الملفات بالترتيب حسب الرقم في اسم الملف

## الملفات

| الملف | الوصف |
|-------|-------|
| `01_initial_setup.sql` | الإعداد الأولي — إنشاء كل الجداول والفهارس وسياسات RLS |

---
> كل تعديل جديد على قاعدة البيانات يُضاف كملف جديد بالتسلسل:
> `02_اسم_التعديل.sql`, `03_اسم_التعديل.sql`, ...
